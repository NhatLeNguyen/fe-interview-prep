# Cert Prep (AWS CLF/SAA + PMP) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans / subagent-driven-development. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Thêm nội dung tự học + câu ôn tập (quiz/flashcard) cho AWS CLF, AWS SAA, PMP; tách khỏi FE bằng trục `track`.

**Tech Stack:** Next 16 App Router, Supabase (RLS), TS, Vitest.

Spec: [`docs/superpowers/specs/2026-07-19-cert-prep-design.md`](../specs/2026-07-19-cert-prep-design.md).
